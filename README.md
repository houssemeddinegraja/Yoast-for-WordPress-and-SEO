## About the Project
This project documents the journey of auditing, optimizing, and managing SEO for live WordPress websites. It bridges the gap between **Web Marketing** (content strategy, keyword targeting, cornerstone content) and **Technical SEO** (HTML meta tags, `<h1>` hierarchies, console auditing, and PHP plugin injection).

---

## Key Topics Covered

### 1. Site Structure & Usability
*   **Categories & Subcategories:** Creating logical, hierarchical buckets for content to help both users and Google bots navigate the site.
*   **WordPress Tags:** Utilizing non-hierarchical descriptive labels (hashtags) to connect related topics.
*   **Internal Linking:** Establishing paths between related posts to boost indexability.

### 2. Cornerstone Content Strategy
*   Identifying and optimizing the 4–5 most critical, evergreen pages on a website.
*   Using Yoast's internal linking tools to pass SEO value (link juice) to cornerstone articles.

### 3. Technical SEO & HTML
*   **Meta Tags:** Managing `<title>` and `<meta name="description">` tags for high CTR in Google SERPs.
*   **Heading Hierarchies:** Enforcing strict, logical outlines using `<h1>`, `<h2>`, and `<h3>` tags.
*   **Image Optimization:** Ensuring `alt=""` attributes are present for accessibility and image search ranking.

### 4. WordPress Plugin Fundamentals
*   Understanding how plugins interact with the WordPress core.
*   Using `add_action()` to hook into the `wp_head` and programmatically inject custom HTML meta tags.
