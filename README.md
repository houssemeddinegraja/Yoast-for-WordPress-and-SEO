# Yoast-for-WordPress-and-SEO

A personal collection of notes, summaries, and takeaways from courses, articles, and other resources I've gone through. Mostly focused on SEO and digital marketing for now, but will expand as I learn more.

## Why this exists

I take a lot of courses and read a lot of material, and it tends to disappear into the void afterward. This repo is my attempt to actually retain and organize it — simplified summaries I can come back to, and that might be useful to others too.

## What's inside

Each file is a simplified, easy-to-read summary of a lesson, course module, or article — not a copy of the original material, just the key ideas explained clearly.

| File | Topic |
|------|-------|
| `seo-basics-summary.md` | How search engines work (crawling, indexing, algorithm, major Google updates) |
| `holistic-seo-summary.md` | Holistic SEO approach (Yoast Academy) |

*(This table will grow as more notes get added.)*

## Notes on format

- Written in plain Markdown, one file per topic/lesson.
- Summaries prioritize clarity over completeness — the goal is to understand the concept quickly, not reproduce the full course.
- Structure may change as the collection grows (e.g., organizing into folders by topic or source).

## Status

Work in progress — this is a living archive that gets updated whenever I finish something worth keeping notes on.
